## [2.0.0] — 2026-07-06

首个开源版本。

### 新增

- **双角色引擎**：同时绑定角色 A + 角色 B，独立角色卡 / System Prompt / 气泡颜色
- **旁听模式**：两个 AI 互相对话，用户旁观看戏
- **身份锚点**：每轮对话末尾注入身份提示，防止角色漂变（可编辑模板）
- **世界书**：关键词触发 + 别名 + 冷却机制 + 优先级排序 + JSON 批量导入导出
- **记忆蒸馏**：自动 / 手动压缩长对话为记忆结晶，替代原始消息参与上下文
- **Galgame 数值引擎**：像素风数值面板（好感度 / 心情 / 警惕度），表里数值分离，4 级 JSON 解析
- **状态书**：吸附在 AI 气泡上的状态卡片，文本模式 + 数值模式双引擎
- **互相认识**：两角色并发观察对方角色卡，自动写入世界书印象条目
- **分支与重试**：消息树状链表，任意节点分支克隆（批量 O(1) I/O），重新生成级联删除 + 复用原消息
- **高级提示词设置**：11 项核心模板可编辑（角色包裹、身份锚点、世界书前缀、蒸馏格式等）
- **加粗变色**：AI 气泡加粗文字按角色着色（charA 翠绿 / charB 紫罗兰 / scribe 琥珀），浅深主题色阶自适应
- **采样参数预设**：🎨 异想天开 / ⚖️ 中规中矩 / 📐 严格规矩 三档
- **SillyTavern V2 角色卡导入**：PNG 隐写 + JSON
- **内置教学预设**：DS 4 PRO 无密钥模型 + 2 个角色卡 + 1 个世界书 + 6 段教学对话（幂等注入）
- **PWA**：Service Worker Cache-First + Web App Manifest，支持离线使用
- **单文件构建**：`npm run build` 产单个 `index.html`，可直接双击打开

### 技术栈

- React 18 + TypeScript 5.5 + Vite 5.4 + vite-plugin-singlefile
- Tailwind CSS 3（`darkMode: 'class'`，全组件双主题适配）
- localForage → IndexedDB（7 个 store + 异步写锁）
- react-markdown
